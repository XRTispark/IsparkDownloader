IsparkDownloader2 插件

一键捕获页面下载链接，发送到下载器进行高速下载

使用步骤

1

浏览网页

访问包含下载链接的页面（GitHub Release、软件下载站、网盘分享等）

2

点击插件图标

点击浏览器工具栏上的 IsparkDownloader2 图标，查看检测到的下载链接

3

选择下载

点击单个链接的下载按钮，或点击"全部下载"批量发送到下载器

功能特性

🔍

自动检测

智能识别页面中的下载链接

📦

多格式支持

支持 EXE、ZIP、PDF、MP4 等常见格式

🌐

GitHub 加速

自动识别 GitHub Release 链接

⚡

一键发送

快速发送到 IsparkDownloader2

注意事项

1. 确保 IsparkDownloader2 桌面应用已安装并运行

2. 插件通过 Native Messaging 与下载器通信，首次使用需要在浏览器中确认权限

3. 如果无法连接到下载器，链接将自动复制到剪贴板

4. 支持 Chrome、Edge、Firefox 等基于 Chromium 的浏览器
